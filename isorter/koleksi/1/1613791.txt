Dukungan debug di Eclipse. Eclipse memungkinkan Anda untuk memulai program Java dalam mode Debug. Eclipse memiliki perspektif Debug khusus yang memberi Anda serangkaian tampilan yang telah dikonfigurasi sebelumnya. Dalam perspektif ini, Anda mengontrol proses eksekusi program Anda dan dapat menyelidiki status variabel.o men-debug program yang Anda perlukan untuk menentukan titik henti sementara. Jika Anda memulai debugger, Eclipse menanyakan apakah Anda ingin beralih ke perspektif Debug setelah titik berhenti tercapai. Jawab Ya dalam dialog yang sesuai. Setelah itu Eclipse membuka perspektif ini, yang terlihat mirip dengan tangkapan layar berikut.
