Dalam cuplikan kode sebelumnya, kita melihat cara mendefinisikan array sederhana bertipe integer. Demikian pula, kita dapat mendefinisikan array jenis apa pun seperti double, karakter, dan string. Dalam C#, array adalah objek. Itu berarti mendeklarasikan array tidak membuat array. Setelah mendeklarasikan array, Anda perlu membuat instance array dengan menggunakan operator baru. Cuplikan kode berikut mendefinisikan array tipe data double, char, bool, dan string.
