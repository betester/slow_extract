NSLOOKUP adalah layanan untuk mencari informasi di DNS (Sistem Nama Domain [RFC1034, RFC1035, RFC1033]). Utilitas NSLOOKUP adalah alat unix. Jika Anda ingin mempelajari lebih lanjut, berikut adalah manual nslookup (halaman manual). Pada dasarnya, DNS memetakan nama domain ke alamat IP. Meskipun layanan online web ini dapat meminta server DNS tertentu, dalam banyak kasus mungkin cukup dan nyaman hanya dengan menggunakan server nama default KLOTH.NET ns.kloth.net atau localhost/127.0.0.1.
