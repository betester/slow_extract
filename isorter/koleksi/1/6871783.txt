Plug-in akan secara otomatis menulis file batch untuk Anda. File batch dapat dipanggil dari prompt perintah, dari file batch lain, Windows Explorer atau Windows Task Scheduler. Perangkat lunak Adobe Acrobat Professional secara otomatis dimulai dan keluar setelah pemrosesan selesai. Dimungkinkan untuk menimpa pilihan file atau folder input/output dalam urutan/tindakan batch Acrobat dan melewati jalur yang berbeda sebagai parameter file batch.ust pilih urutan batch yang ada (tindakan) dari daftar dan perangkat lunak akan melakukan sisanya. AutoBatch akan secara otomatis menulis file batch yang siap digunakan. Peluncuran file batch ini akan memulai Adobe Acrobat Professional dan memulai eksekusi urutan batch yang dipilih.
