Citrix XenApp dan Layanan Desktop Jarak Jauh memungkinkan aplikasi Windows dan sumber daya komputasi dikelola secara terpusat di pusat data yang aman. Pengguna dapat mengakses aplikasi dari mana saja dan dari klien non-Windows. Lingkungan XenApp terdiri dari tiga bagian: 1 Sistem operasi multi-pengguna: Microsoft Windows Server dengan fitur Remote Desktop Session Host memungkinkan banyak pengguna mengakses server secara mandiri.
