Ketika saya membuat subdomain di godaddy.com, satu-satunya pilihan adalah meneruskan subdomain itu, dan saya tidak ingin meneruskannya, saya ingin mengarahkannya ke server lain. Saya tidak ingin situs utama atau akun email ke terganggu baik. aktif suara tertua. suara naik 3 suara turun. Saya pikir Anda mencoba mengatur catatan Cname di mana Anda perlu mengatur catatan A lain dan mengarahkannya ke IP server. Anda tidak dapat menggunakan CNAME karena itu dirancang untuk menunjuk ke domain lain, bukan menunjuk ke alamat IP, dan Anda mencoba untuk menunjuk ke IP.
