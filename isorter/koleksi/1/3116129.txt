VOB adalah singkatan dari DVD Video Object dan merupakan file inti dari DVD yang berisi data film aktual, aliran film MPEG, dan audio yang didukung. Untuk memutar file VOB, Anda harus memiliki pemutar DVD yang mendukung format VOB. Namun, jika Anda memiliki Windows Media Player (versi 12 atau lebih tinggi) atau VLC Media Player, cukup baik untuk memutar file tersebut. untuk mendapatkan file VOB dari DVD karena alasan tertentu, cukup buka DVD di Windows Explorer alih-alih memutarnya melalui pemutar media. File VOB kemungkinan besar berada di bawah folder VIDEO_TS.
